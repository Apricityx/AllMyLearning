# 网络层

## NAT技术

用于在局域网与广域网之间转换IP地址，一般由路由器实现。

### 路由协议分类

内部网关协议（IGP, Interior Gateway Protocol）

- OSPF

首先通过邻居发现建立拓扑图，邻接建立后，会交换链路状态数据库中的信息(例如距离成本之类的)。接下来路由器会建立一个拓扑结构，基于拓扑图，使用迪杰斯特拉算法计算最短路径，最后将计算结果构建在路由表中，用于转发。

外部网关协议（EGP, Exterior Gateway Protocol）

用于AS之间（如ISP与ISP之间）。

策略性路由：可以根据政策选择路径，而非仅依据最短路径。

比IGP更复杂，收敛速度较慢，但扩展性强，适合互联网规模。

支持路径属性、环路避免、负载均衡等特性。

- BGP（Border Gateway Protocol）：边界网关协议

## ICMP

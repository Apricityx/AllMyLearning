# 神经网络

神经网络分为多层，每一层有他自己的作用

训练一个模型的实质实际上是猜测权重与偏置

使用识别数字举例，神经网络可以分为4层，第一层负责将一个n*n的图像转换为n^2个神经元，每一个神经元乘上权重并经过偏置的调整转换为一个评分


## 损失函数（交叉熵）
$$L = -\frac{1}{N} \sum_{n=1}^{N} \sum_{c=1}^{C} y_{n,c} \log(p_{n,c})$$

这个函数用于衡量模型的预测值与真实值之间的差距

该函数的大致意义是将每个样本对于预测准确实际值的概率取对数函数累加起来再取平均值，L越小越好

Cost Function
输入：weights / biases
输出：a single number (cost), bigger is worse

对于一个很多参数的代价函数，不能采用传统的求导的方式获取一个极值点，取而代之的方法是取一个随机点，然后判断这个点应该往什么方向移动

计算方法：
1. 计算坡度
2. 向坡度指向的方向跨出一小步

$$ \vec{w} = [w_1,w_2,w_3....]$$
$$ - \Delta C \vec{w} = [\textcolor{red}{0.3},\textcolor{red}{0.4},\textcolor{aqua}{1.3}....]$$

0.3 0.4 小于1的值表示$w_1 w_2$应当调小,而大于1的值 表示$w_3$应当调大

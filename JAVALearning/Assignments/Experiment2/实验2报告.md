# 2 类图

使用plantuml语法绘制Node、Edge、Entity、Relationship、KnowledgeGraph等5个核心类的类图，以及它们之间的关系。

```plantuml
@startuml
package oop.task.exp.e02.kg.core {

    class KnowledgeGraph {
        - Set<Entity> entities
        - Set<Relationship> relations
        - Map<Entity, Set<Relationship>> outMap
        - Map<Entity, Set<Relationship>> inMap
        - int seqNumber
        
        + KnowledgeGraph()
        + Optional<Entity> addEntity(String type, String name, String attribute)
        + boolean addEntity(Entity e)
        + Optional<Relationship> addRelation(Entity from, Entity to)
        + void removeEntity(Entity e)
        + void removeAllRelsByFrom(Entity from)
        + void removeAllRelsByTo(Entity to)
        + Set<Entity> getEntities()
        + Set<Relationship> getRelations()
    }

    class Entity extends Node {
        - String name
        - String type
        - String attribute
        
        + Entity(int seqNumber, String type, String name, String attribute)
        + String getType()
        + String getAttribute()
        + void setName(String name)
        + String getName()
        + void setAttribute(String text)
        + void setType(String text)
    }

    class Relationship extends Edge {
        + String name
        + String attribute
        
        + Relationship(Entity from, Entity to)
        + String getName()
        + String getAttribute()
        + void setName(String text)
        + void setAttribute(String text)
    }

    class Edge {
        - Node from
        - Node to
        
        + Edge(Node from, Node to)
        + Node getFrom()
        + void setFrom(Node from)
        + Node getTo()
        + void setTo(Node to)
        + boolean equals(Object o)
        + int hashCode()
    }

    class Node {
        - int id
        
        + Node(int id)
        + int getId()
        + void setId(int id)
        + boolean equals(Object o)
        + int hashCode()
    }

    KnowledgeGraph --> Entity
    KnowledgeGraph --> Relationship
    Relationship --> Entity : from, to
    Entity --> Node
    Relationship --> Edge
    Edge --> Node
}

@enduml
```

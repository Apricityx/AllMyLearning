# 复习笔记

## 小数进制转换

先处理整数部分，再处理小数部分。

### 16进制转10进制
例：`1A3.32H`转换为10进制。
整数部分：3*16^0 + 10*16^1 + 1*16^2 = 419
小数部分：3*16^-1 + 2*16^-2 = 3/16 + 2/256 = 0.1875

### 10进制转16进制
例：`419.1875`转换为16进制。
整数部分：
419/16 = 26...3
26/16 = 1...10
1/16 = 0...1
所以整数部分为`1A3`
小数部分：
0.1875 * 16 = 3

## 标志位

1. CF: Carry Flag，进位标志位 进位时为1
2. PF: Parity Flag，奇偶标志位 二进制数1的个数为偶数时为1
3. AF: Auxiliary Carry Flag，辅助进位标志位 低四位进位时为1
4. ZF: Zero Flag，零标志位 结果为0时为1
5. SF: Sign Flag，符号标志位 结果为负数时为1
6. OF: Overflow Flag，溢出标志位 结果超出范围时为1
